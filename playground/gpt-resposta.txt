Para que as divs "sidebar" e "produtos" fiquem lado a lado, você precisa garantir que ambas tenham a mesma altura. Como as suas divs estão configuradas com `display: inline-block`, elas devem estar corretamente alinhadas, mas se uma delas tiver conteúdo que faz com que a altura da div seja maior, isso pode causar problemas de alinhamento vertical.

Aqui estão algumas sugestões para resolver o problema:

1. **Definir altura para as divs**: Se você garantir que ambas as divs tenham a mesma altura, isso pode ajudar. Você pode usar `height: 100%` ou definir uma altura fixa.

2. **Usar flexbox**: Uma maneira mais moderna e flexível de alinhar elementos lado a lado é usar flexbox. Modifique o estilo da classe `.main` para usar `display: flex;`. Isso fará com que as divs se alinhem automaticamente.

Aqui está um exemplo de como você pode alterar seu CSS para usar flexbox:

```css
.main {
    margin: 0 auto;
    max-width: 1200px;
    padding: 40px 0px;
    display: flex; /* Adiciona Flexbox */
}

.sidebar {
    margin-right: 50px;
}

.produtos {
    flex-grow: 1; /* Faz com que a div produtos ocupe o espaço restante */
}
```

3. **Ajustes no CSS**: Após aplicar as mudanças, você pode precisar ajustar o `margin` e o `padding` para garantir que o layout fique como você deseja.

Com essas mudanças, as divs "sidebar" e "produtos" devem ficar alinhadas lado a lado corretamente. Se você continuar a ter problemas, verifique se há alguma regra de CSS que possa estar afetando o layout.
